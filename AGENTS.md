# Bitcraft Brand Assets

Static asset repository - no build system.

## Structure

- `logo/` - SVG sources + PNG/favicon exports
- `colors/palette.md` - Color specs with CSS variables
- `typography/fonts.md` - Font specs (Inter, JetBrains Mono)
- `templates/` - HTML templates
- `brandbook/bitcraft-brandbook.html` - Interactive brand guide

## Colors

- Dark Olive: `#556B2F`
- Olive Drab: `#6B8E23`
- Forest Green: `#228B22`
